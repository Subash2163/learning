HTML - HyperText Markup Language

HTML, short for HyperText Markup Language, is the foundation of the World Wide Web. It's a markup language used to structure and organize content on web pages. HTML defines the structure of a webpage, telling a web browser how to display text, images, links, and other multimedia elements.

CSS - Cascading Style Sheet

CSS stands for Cascading Style Sheets. It's a stylesheet language used to describe the look and formatting of a document written in HTML (or XML). While HTML provides the structure of the page, CSS controls the presentationâ€”how it looks to users.

Bootstrap

Bootstrap is a free and open-source front-end framework used to develop responsive and mobile-first websites and web applications. It was originally developed by Twitter and is now maintained by the open-source community. Bootstrap provides a set of ready-to-use CSS and JavaScript components that help developers build clean, consistent, and responsive web interfaces quickly.
